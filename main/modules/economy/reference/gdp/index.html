    GDP.Average_GDP¶

    stock

    Units: b$/year · Dimensions: Run · Inflows: GDP.change_in_average_GDP

    Average GDP is a smoothed stock, a running average of GDP over time.

    Many behaviors depend on trend GDP, not instantaneous GDP Smoothing removes noise from year-to-year shocks.

    Nominal GDP = “What is happening this year?” Average GDP = “What is happening on average over the past few years?”

    GDP.initial_average_GDP

    GDP.Cumulative_Discount_Exponent¶

    stock

    Units: dmnl · Dimensions: Run · Inflows: GDP.discount_rate_accumulation

    The stock is just a memory of how much discounting has accumulated so far. Because safe interest rate is dynamic, it changes over time in the model. This stock answers: “How much total discounting has built up since 2030/ NPV start year?” The initial value is 0 because before policy start year/ 2030, we do not want any accumulated discounting yet.

    GDP.instantaneous_welfare_from_real_GDP¶

    aux

    Units: dmnl · Dimensions: Run

    This variable converts GDP into a welfare score. Each year’s welfare is the log of real GDP in 2021 prices (diminishing marginal welfare of extra GDP). Higher GDP still means higher welfare, but extra GDP has smaller additional value.

    Equation:

    GDP.welfare_discount_factor¶

    aux

    Units: dmnl · Dimensions: Run

    This tells the model how much to discount future welfare each year, using the safe interest rate from the Government sector. This gives the shrinking weight over time: at 2030, factor = 1. Later, factor gets smaller so later welfare counts less.

    Equation: 

    EXP(-GDP.Cumulative_Welfare_Discount_Exponent)

    energy_investments.adaptive_energy_investments_proportional_factor¶

    aux Internal calibration parameter

    Units: dmnl · Dimensions: Run · Range: -50 – 0

    The larger the P factor is, the stronger the controller reacts to differences between the set point value and the actual value (0 relative supply demand missmatch). Negative values of P causes compensation of differences, positive values would cause amplification of differences.

    Equation:
